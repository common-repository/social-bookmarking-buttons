== Description ==

This lightweight plugin adds sharing buttons at the end of each post. HTML validated and fully cross-browser compatible (including Internet Explorer). This is the only social bookmarking plugin that displays different content based on the browser.

Includes:<ul>
<li>Facebook's Like button</li>
<li>Twitter's Tweet/Retweet button</li>
<li>Digg's Digg button</li>
<li>StumbleUpon's Stumble button</li>
</ul>
Each social bookmarking button includes a real-time count of Tweets, Diggs, or Likes.


== Installation ==

1. Upload `Social-Bookmarking-Buttons` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy :)
